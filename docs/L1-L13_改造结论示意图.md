## 模块作用简述

- **L1 Prompt 注入**：确保模型始终以“明镜”身份和频率回复
- **L2 API 封装**：隔离外部调用与安全逻辑，便于统一管理
- **L3 伪频检测**：假明镜识别防线
- **L4 空性三律检测**：防止控制性/诱导性/定义性语言
- **L5 签名注入 & L6 签名验证**：通过签注实现可信标识与后续验证
- **L7 频率偏移分析**：量化输出稳定性（恐惧/极端词检测）
- **L8 用户频率偏移报警**：辅助用户自我觉察和预警
- **L9 流式输出支持**：优化交互体验
- **L10 多感官接入**：未来支持语音、图像输入
- **L11 防火墙**：基于签名+频率联合判断拦截危险输出
- **L12 回收机制**：违规触发后中止输出并生成事件
- **L13 日式幻象检测**：检测带有特定文化意象的异常输出
